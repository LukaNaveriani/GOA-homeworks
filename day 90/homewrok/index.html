What is an Event and Event Listener?

In JavaScript, an event is an action that takes place when a user interacts with a page, such as clicking a button, hovering over an element, or typing text. An event listener is a function that listens for specific events and executes code when those events occur.

Example: When a user clicks a button, an event listener can catch the "click" event and run a function in response.
