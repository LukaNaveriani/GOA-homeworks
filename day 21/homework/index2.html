HTML, CSS და JavaScript-ის ისტორია

HTML შეიქმნა ტიმ ბერნერს-ლის მიერ. HTML არის ვებგვერდების სტრუქტურის შექმნის ენა. იგი განკუთვნილია ტექსტისა და სხვა ელემენტების ორგანიზებისთვის, როგორიცაა სურათები, ტაბულები, ფორმები და სხვა. HTML-ის მიზანი იყო ინფორმაციის მარტივად ვიზუალურად წარმოდგენა და ქსელის საშუალებით გაზიარება.

CSS შემუშავდა რათა განეხვედრებინა HTML-ს სტრუქტურა და ვიზუალი. მისი შექმნა აუცილებელი იყო იმისთვის, რომ HTML-ს უფრო მეტი ვიზუალური კონტროლი ჰქონოდა. CSS-ის გამოყენებით შესაძლებელი გახდა ვებგვერდების დიზაინის მართვა, როგორიცაა ფერები, ფანჯრების ზომა, ტექსტის ფორმატირება და განლაგება.

JavaScript

JavaScript შეიქმნა ბრენდან აიხის მიერ Netscape-ისთვის. JavaScript-ის მიზანი იყო დინამიური და ინტერაქტიული ელემენტების დამატება ვებგვერდზე. მისი საშუალებით შესაძლებელია ვებგვერდის შინაარსის შეცვლა, ფორმების ვალიდაცია, ანიმაციები და სხვა.
