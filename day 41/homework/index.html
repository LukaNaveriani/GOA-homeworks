Front-end განვითარება ეხება ვებ-გვერდების შექმნის მხარეს, რომელიც მომხმარებელთან ურთიერთობს. ეს მოიცავს HTML-ს, CSS-ს და JavaScript-ს.

What is Frontend?

Front-end არის ვებ-გვერდის განსახიერება, რომელიც მომხმარებელს აჩვენებს. ეს არის ის ნაწილი, რომელიც ვებ-ბრაუზერში ჩანს. Front-end-ს ქმნის HTML, CSS და JavaScript. HTML წარმოადგენს სტრუქტურას, CSS ამ სtructure-ს სტილიზაციას აკეთებს, ხოლო JavaScript კი ინტერფეისს ფუნქციონალობას სძენს.

HTML

HTML (HyperText Markup Language) არის საბაზისო ენით ვებ-გვერდების სტრუქტურის შექმნა. ის განსაზღვრავს ვებ-გვერდის სხვადასხვა კომპონენტებს, როგორიცაა თავები, პათები, სურათები და სხვა.

History of HTML

HTML შეიქმნა 1991 წელს ტიმ ბერნერს-ლიმის მიერ, რომელიც მუშაობდა ვებ-ტექნოლოგიებზე. მისი მიზანი იყო ერთიანი ფორმატის შექმნა სხვადასხვა კომპიუტერებს შორის ინფორმაციის გაზიარებისთვის.

CSS

CSS (Cascading Style Sheets) არის ენა, რომელიც გამოიყენება HTML ელემენტების სტილიზაციისთვის. CSS განსაზღვრავს ფერებს, შრიფტებს, განლაგებას და სხვა ვიზუალურ მახასიათებლებს.

History of CSS

CSS-ის შექმნა დაიწყო 1994 წელს, რათა HTML დოკუმენტების სტილი გამოხატულიყო ცალკე, რაც დაეხმარა დიზაინერებს და დეველოპერებს.

JavaScript

JavaScript არის პროგრამირების ენა, რომელიც გამოიყენება ვებ-გვერდის ფუნქციონალობის დასამატებლად. ის საშუალებას აძლევს გვერდს დინამიურად შეიცვალოს მონაცემები და ინტერაქტიულობა დაემატოს.

History of JavaScript

JavaScript შეიქმნა 1995 წელს ბრენდან აიჩის მიერ და გამოყენება დაიწყო როგორც მყარი ილუსტრაცია, რომლითაც ინტერაქტივობა იყო შესაძლებელი ვებ-გვერდებზე.
